## Â§4.3 Vegetable Substrate Identification

Every piece of produce has a PLU (Price Look-Up) sticker. Here's how to read them:

| Code Format | Meaning |
|-------------|---------|
| 4-digit (e.g., 4065) | Conventionally grown |
| 5-digit starting with **9** (e.g., 94065) | Organically grown |
| 5-digit starting with **8** (e.g., 84065) | Genetically modified (rarely used) |

> [!success] Procurement Rule
> **Organic (9xxxx) is nice to have, not required.**
> 
> If the organic version isn't available or costs significantly more, conventional (4-digit) is perfectly fine for all protocols.

See [[PLU Reference]] for a complete code matrix.
