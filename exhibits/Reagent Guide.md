### Reagent D â€” [[Doenjang]]

| Parameter | Specification |
|-----------|---------------|
| **Common Name** | Doenjang (ëœìž¥) |
| **English** | Fermented soybean paste |
| **Flavor** | Salty, earthy, deeply savory (umami), slightly funky |
| **Texture** | Brown paste, coarse with visible bean fragments |
| **Storage** | Refrigerate after opening, 12+ months |
| **Used In** | [[Protocol A - Aqueous Phase Extraction\|ATP-A]] |

> [!warning] Not the Same as Miso
> Japanese miso looks similar but tastes different. Doenjang is stronger, earthier, and made only from soybeans. Miso includes rice or barley and is milder.
> 
> For AL-ATP-001, use doenjang. Miso is not a valid substitute.
